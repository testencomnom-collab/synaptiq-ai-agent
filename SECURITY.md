# Security Policy

## Supported Versions

Currently, only the latest release of the Synaptiq AI Agent is supported with security updates. 

| Version | Supported          |
| ------- | ------------------ |
| 18.x.x  | :white_check_mark: |
| < 18.0  | :x:                |

## Reporting a Vulnerability

We take the security of our users very seriously. If you discover a security vulnerability within the Synaptiq AI Agent, please **DO NOT** disclose it publicly via GitHub Issues. 

Instead, please use the private **GitHub Security Advisory** feature on our repository to report the issue directly to the maintainers, or email the project maintainer directly.

### What to include
When reporting an issue, please provide as much information as possible:
* The version of the app you are running.
* The steps to reproduce the vulnerability.
* A description of the impact and potential exploit scenarios.

### Expected Response
You can expect an acknowledgment of your report within 48 hours. We will keep you updated on the progress of our investigation and mitigation efforts. Once the issue is resolved, we will publish a coordinated security advisory.
