### What's new
- **Zero CPU Thrashing**: Accessibility Service now uses an intelligent 800ms throttle, drastically reducing CPU usage and eliminating UI freezing during intensive automations.
- **Race Condition Fixes**: Core architecture is now fully thread-safe. Fixed critical double-checked locking issues in LLMAgentService and AgentEngineManager that caused Memory Leaks and OOM crashes.
- **Android 14+ FGS Fix**: Explicitly added `PROPERTY_SPECIAL_USE_FGS_SUBTYPE` preventing `SecurityException` crashes for Foreground Services.
- **CI/CD Stabilization**: Fixed missing executable bits on the build scripts preventing CodeQL analysis workflows from completing.
