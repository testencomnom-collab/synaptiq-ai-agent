### What's new
- **Auto-Scroll Navigation**: The agent can now automatically scroll through apps and menus using SCROLL_DOWN and SCROLL_UP.
- **Memory Leak Fixed**: The AgentAccessibilityService correctly releases accessibility nodes after reading them.
- **15-Second Timeout Fix**: The UI action loop now has a timeout.
- **Improved Network Stability**: Connection drops no longer hard-crash the task.
- **Code & UI Cleanup**: Obsolete UI elements and dead code have been cleaned up.
